# IntVector的意义
IntVector是一个自动扩容的数组，展示了如何对原始数组进行操作。
由于内存就是一条线性的字节序列，数组是最贴近内存特点的数据结构。
对数组操作，其实就是对内存操作。

# TestIntVector
